# 明確性の原則：5W1H+Role

| 要素 | 内容 | 具体例 |
|------|------|--------|
| **Role** | AIに求める専門性 | 簿記教師、キャリアカウンセラー |
| **Who** | 対象者・読み手 | 高校1年生、保護者、同僚教師 |
| **What** | 作成する内容 | 学習指導案、練習問題、説明資料 |
| **When** | 時期・タイミング | 新学期、中間試験前、進路指導時期 |
| **Where** | 使用場面 | 教室、実習室、保護者会 |
| **Why** | 目的・理由 | 理解促進、動機付け、技能習得 |
| **How** | 方法・形式 | 講義形式、グループワーク、個別指導 |

# 文脈提供の重要性

### 提供すべき文脈情報
- **教育環境**：学校の特徴、生徒の特性、地域性
- **学習状況**：既習内容、習熟度、学習目標
- **制約条件**：時間、設備、予算等の現実的制約
- **期待する成果**：最終的に達成したい目標

```
【背景情報の例】
本校は地方の商業高校で、卒業生の約70%が地元企業に就職します。
生徒は実践的な学習を好み、将来の職業との関連を重視する傾向があります。
現在、簿記2級取得を目指すクラスで、工業簿記の学習に入ったところです。
```

# チェーン・オブ・ソート（思考の連鎖）

複雑な問題を段階的に分析・解決する技術

```
商業高校生の進路選択について、以下のステップで分析してください：

ステップ1: 現在の労働市場の状況を分析
ステップ2: 商業高校生に求められるスキルを特定
ステップ3: 具体的な進路選択肢を提示
ステップ4: 各選択肢のメリット・デメリットを比較
ステップ5: 最適な進路選択の指針を提案
```

### 効果
- 論理的な思考プロセスの可視化
- 複雑な問題の段階的解決
- 学習者の理解促進

---

# ロールプレイング技術

AIに特定の専門家の役割を与えて、専門性の高い回答を得る

### 商業教育での専門家役割例
- 公認会計士　- 中小企業診断士
- キャリアコンサルタント　- マーケティング専門家

```
地元企業でのインターンシップ導入について、
以下の立場からそれぞれ意見を述べてください：
1. 高校教師の立場
2. 企業経営者の立場  
3. 生徒の立場
4. 保護者の立場
```

---

# 制約条件の活用

適切な制約を設けることで、実用的な成果物を得る

### 制約の種類
- **分量制約**：「300文字以内」「A4用紙1枚」
- **時間制約**：「5分で説明可能」「50分授業用」
- **形式制約**：「箇条書きで」「表形式で」

```
簿記の「減価償却」について、
高校1年生が10分で理解できる説明を、
身近な例を使って、
300文字以内で、
会話形式で作成してください。
```
